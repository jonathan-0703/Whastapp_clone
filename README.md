# 📱 Whatsapp Clone

Proyecto Final del Bootcamp enfocado en el desarrollo de una aplicación tipo mensajería inspirada en WhatsApp.

---

## 🚀 Descripción

Este proyecto consiste en la construcción de un sistema de mensajería que permite:

* Gestión de usuarios
* Envío de mensajes
* Bloqueo de usuarios
